## 추천 구조

```txt
GitHub README
   ↓
Portfolio Website 버튼
   ↓
GitHub Pages 포트폴리오 사이트
```

이 방식이 가장 깔끔하고 실제 개발자 포트폴리오에서 많이 사용하는 구조입니다.

---

# 1. GitHub Repository 생성

추천 이름:

```txt
portfolio
```

또는:

```txt
yeongnam-portfolio
```

---

# 2. 파일 구조

```txt
portfolio/
 ├─ index.html
 ├─ style.css
 ├─ script.js
 └─ assets/
```

---

# 3. GitHub Pages 활성화

GitHub Repository → Settings → Pages

아래처럼 설정:

```txt
Branch : main
Folder : /(root)
```

저장 후:

```txt
https://YOURNAME.github.io/portfolio
```

형태로 사이트가 생성됨.
